# Animation

CSS 애니메이션은 CSS 트랜지션과 동일한 방식으로 적용됩니다. 차이점 v-enter 클래스가 요소가 삽입 된 직후가 아닌 animationend 이벤트에서 제거됩니다.



CSS 애니메이션이 반영된 아래 데모 코드를 실행해보세요.

Scale Up

데모 코드는 다음과 같습니다. animation 속성 또한 활성화 클래스에 설정합니다.




 















 


 



 
 
 
 
 
 
 
 
 
 
 































<template>
  <div class="demo-transition">
    <button @click="isVisible = !isVisible">애니메이션 토글</button>
    <transition name="scale-up">
      <div class="box" v-if="isVisible">Scale Up</div>
    </transition>
  </div>
</template>

<script>
export default {
  name: "DemoBasicAnimation",
  data: () => ({
    isVisible: true
  })
};
</script>

<style scoped>
.scale-up-enter-active {
  animation: scale-up 0.6s cubic-bezier(0.2, 0.01, 0.57, 1.38);
}
.scale-up-leave-active {
  animation: scale-up 0.6s cubic-bezier(0.2, 0.01, 0.57, 1.38) reverse;
}

@keyframes scale-up {
  0% {
    transform: scale(0);
  }
  50% {
    transform: scale(1.6);
  }
  100% {
    transform: scale(1);
  }
}

.box {
  width: 150px;
  padding: 20px;
  margin: 20px auto 0;
  background: #3dbd83;
  color: #fff;
  font-weight: bold;
}

button {
  cursor: pointer;
  border: 1px solid #ccc;
  border-radius: 4px;
  background: #fff;
  padding: 1em;
  color: #434343;
  font-size: 12px;
  transition: all 0.3s ease-in;
}
button:hover,
button:focus {
  border-color: currentColor;
}
button:focus {
  outline: none;
}
button:active {
  transform: scale(0.95);
}
</style>
