# 테스트

Vue CLI 도구는 Jest 또는 Mocha + Chai와 같은 테스트 도구를 사용한 유닛 테스트 유틸리티 라이브러리 Vue Test Utils을 포함하고 있습니다. Jest를 사용한 Vue 애플리케이션 테스팅 방법에 대해 학습해보도록 합시다.


# 시작하기

Vue CLI 도구 create 명령을 사용해 테스트를 실습할 프로젝트를 생성합니다.

Manually select features > Unit Testing > Jest 선택 후 설치

$ vue create vue-test-practice
