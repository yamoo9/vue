# 재귀 컴포넌트란?

재귀 컴포넌트를 공부하기에 앞서 재귀 함수에 대해 이해하고 있어야 합니다.

# 팩토리얼 함수

재귀 함수의 대표적인 예제인 팩토리얼 함수로, 전달된 수보다 작거나, 같은 모든 양의 정수의 곱을 반환합니다.

const factorial = n => {
  if ( n <= 0) { return 1; }
  return n * (n > 1 ? factorial(n - 1) : 1);
};

factorial(3); // 3 * 2 * 1 = 6
factorial(5); // 5* 4 * 3 * 2 * 1 = 120

# TreeView 컴포넌트

재귀 컴포넌트의 주된 사용 예시로 트리뷰 컴포넌트를 들 수 있습니다. 트리뷰 아이템 컴포넌트는 컴포넌트 자신을 호출해 재사용해 트리 뷰를 구성합니다. 아래 예제의 메뉴 버튼을 클릭해보세요. 트리 뷰가 확장/축소 됩니다.

                아래 코드는 TreeViewItem.vue의 템플릿입니다. 템플릿 코드 내에서 컴포넌트 자신을 호출하여 재 사용하고 있습니다. 전달된 tree 객체가 subTree 속성을 포함할 경우 재귀적으로 호출 실행됩니다.

                TreeViewItem.vue











                 




                <template>
  <li>
    <button @click="toggle" :title="label">
      {{ tree.title }}
      <span
        v-if="hasSubTree"
        :aria-label="label"
      >{{ open ? '▾' : '‣' }}</span>
    </button>
    <ul class="treeView" v-show="open">
      <li is="tree-view-item" v-for="(tree,i) in tree.subTree" :key="i" :tree="tree"></li>
    </ul>
  </li>
</template>
