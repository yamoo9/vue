# ESLint Plugin Vue

eslint-plugin-vue 플러그인은 Vue.js 공식 ESLint 플러그인 입니다. 이 플러그인은 Vue 파일 <template>, <script> 코드를 체크해 잘못된 문법, 잘못 사용된 디렉티브, Vue.js 스타일 가이드에 어긋난 것을 찾아 오류를 화면에 표시합니다.

# Playground

ESLint Plugin Vue가 적용되어 에디터 화면에 오류가 표시된 플레이그라운드를 통해 오류 해결 과정을 실습할 수 있습니다.
