# 리디렉션 (항로 변경)

강의 경로(path)를 /lecture에서 /lecture-us로 변경한 경우, redirect 속성을 사용해 /lecture 경로로 라우트 수정하여 이동하도록 설정할 수 있습니다. 사용자가 /lecture 경로로 접근할 경우, Lecture 라우트로 리디렉션 되어 /lecture-us 경로로 페이지가 화면에 표시됩니다.

rouer.js





 



new Router({
  routes: [
    // ...
    { name: 'Lecture', path: '/lecture-us', component: Lecture },
    { path: '/lecture', redirect: { name: 'Lecture' } },
  ]
})

# 알리아스 (별칭)

또 다른 방법으로 별칭(alias)를 등록하는 방법이 있습니다. 앞서 다룬 리디렉션과 다른 점은 별칭으로 등록된 경로(path)가 그대로 유지된다는 점입니다. 즉, /lecture, /lecture-us 경로 모두 동일한 페이지를 화면에 표시합니다.






 




new Router({
  routes: [
    // ...
    {
      name: 'Lecture', path: '/lecture', component: Lecture,
      alias: '/lecture-us'
    }
  ]
})
