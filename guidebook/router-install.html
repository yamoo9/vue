# 설치

클라이언트 환경에서 라우팅을 사용하기 위해 프로젝트에 Vue Router 라이브러리를 설치합니다.

 


$ yarn add vue-router
# npm i vue-router

2

NOTE

Vue CLI 3 도구 add 명령을 사용해 Vue Router를 프로젝트에 추가할 수도 있습니다. 하지만 이 방법은 데모 페이지를 만들어 내므로, 매번 데모 페이지를 제거하는 수고를 하고 싶지 않을 경우 직접 Vue Router를 구성하는 것이 좋습니다.

# 설정

Vue Router 모듈을 로드, Vue 플러그인으로 사용 설정합니다. views 디렉토리 내부의 페이지 컴포넌트를 로드하여 routes 배열에 등록합니다. 등록되는 라우트 아이템 객체는 name, path, component 속성을 가집니다.

router.js

import Vue from 'vue';

// Vue Router 모듈 로드
import Router from 'vue-router';

// 페이지 컴포넌트 로드
import Home from '@/views/Home.vue';
import Lecture from '@/views/Lecture.vue';

// Vue Router 플러그인 사용 설정
Vue.use(Router);

// Vue Router 인스턴스 생성, 내보내기
export default new Router({
  // 라우트 등록
  routes: [
    { name: 'Home', path: '/', component: Home },
    { name: 'Lecture', path: '/lecture', component: Lecture },
  ]
});

main.js

// Vue Router 인스턴스 로드
import router from './router';

// Vue 인스턴스에 Vue Router 인스턴스 추가
new Vue({
  router,
  // ...
});
