# 라우터 컴포넌트

Vue 애플리케이션이 싱글 페이지 애플리케이션(SPA)으로 동작하려면 네이티브 HTML <a> 요소 대신, Vue Router가 제공하는 <router-link>, <router-view> 컴포넌트를 사용해야 합니다.

App.vue







 
 



 




<template>
  <div id="app">

    <!-- 내비게이션 -->
    <nav class="gnb">
      <!-- 라우트 Link -->
      <router-link to="/">홈</router-link>
      <router-link to="/lecture">강의</router-link>
    </nav>

    <!-- 라우트 View -->
    <router-view/>

  </div>
</template>

# 이름으로 라우트 연결

라우트 패스(path) 대신 이름(name)을 RouterLink 컴포넌트의 to 속성 값으로 설정할 수 있습니다.

<router-link :to="{ name: 'Home' }">홈</router-link>
<router-link :to="{ name: 'Lecture' }">강의</router-link>
1
2

주의!

JavaScript 식(Expression)을 사용하려면 v-bind 디렉티브를 사용해야 합니다. (to ➪ :to)

라우트 객체의 name 속성 값과 일치하는 경로의 페이지가 화면에 표시됩니다.






 




 






// router.js

export default new Router({
  routes: [
    {
      name: 'Home',
      path: '/',
      component: Home
    },
    {
      name: 'Lecture',
      path: '/lecture',
      component: Lecture
    },
  ]
})

NOTE

라우트 이름(name)을 사용하면 추후 URL 패스(path)가 변경될 경우, RouterLink 컴포넌트를 찾아 수정할 필요 없이 router.js 파일에서 일괄적으로 관리할 수 있어 유용합니다.
