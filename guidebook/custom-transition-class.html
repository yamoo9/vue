# Custom Transition Class

다음 속성을 제공하여 사용자 정의 변환 클래스를 지정할 수도 있습니다. 이 속성은 기존의 클래스 이름을 덮어씁니다.

NOTE

커스텀 트랜지션 클래스는 CSS 애니메이션 라이브러리를 사용할 때 유용합니다.

  • enter-class
  • enter-active-class
  • enter-to-class
  • leave-class
  • leave-active-class
  • leave-to-class




CSS 애니메이션 라이브러리 CSS3 Animation Cheat Sheet를 사용하는 데모를 살펴보겠습니다.

PULLUP

먼저 라이브러리 (animations.css)를 호출해야 합니다. 데모에서는 컴포넌트 내부에서 라이브러리를 호출했습니다.

<style src="./animations.css"></style>
1

컴포넌트 템플릿 코드를 살펴보면 enter-active-class 커스텀 트랜지션 클래스를 적용해 동적으로 animationType 값을 변경합니다. animationType은 버튼을 클릭하면 버튼에 연결된 컴포넌트 메서드에 의해 변경됩니다.





 




<button @click="pullUp">pullUp 애니메이션</button>
<button @click="pullDown">pullDown 애니메이션</button>

<transition
  :enter-active-class="animationType"
>
  <div class="box" v-show="isVisible">{{ animationText }}</div>
</transition>

컴포넌트 로직 코드는 다음과 같습니다. 버튼 클릭 이벤트에 따라 동적으로 변경될 데이터 animationType을 추가하고, 각 버튼에 연결될 메서드를 작성합니다.




 











 
 
 
 
 
 
 
 
 
 
 
 
 
 



export default {
  name: "DemoLibAnimation",
  data: () => ({
    animationType: "pullUp",
    isVisible: true
  }),
  computed: {
    animationText() {
      return this.animationType.toUpperCase();
    }
  },
  methods: {
    resetVisibleState() {
      this.isVisible = false;
    },
    pullUp() {
      this.resetVisibleState();
      this.animationType = "pullUp";
      setTimeout(() => {
        this.isVisible = true;
      }, 0);
    },
    pullDown() {
      this.resetVisibleState();
      this.animationType = "pullDown";
      setTimeout(() => {
        this.isVisible = true;
      }, 0);
    }
  }
};
