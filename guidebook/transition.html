# Transition

Vue 애플리케이션은 요소가 DOM에 삽입, 업데이트 또는 제거 될 때 트랜지션을 적용하는 다양한 방법을 제공합니다.

  • CSS Transition, Animation 클래스 속성을 자동으로 적용합니다.
  • 트랜지션 훅을 사용해 JavaScript에서 DOM을 직접 조작할 수 있습니다.
  • CSS 애니메이션 라이브러리를 활용할 수 있습니다.
  • JavaScript 애니메이션 라이브러리를 활용할 수 있습니다.




NOTE

# 트랜지션 컴포넌트

Vue는 트랜지션을 위한 컴포넌트 <transition>을 제공 합니다. 트랜지션 컴포넌트는 요소 또는 컴포넌트를 enter → leave 흐름으로 트랜지션 효과를 적용합니다.

  • 조건부 렌더링 v-if
  • 조건부 표시 v-show
  • 동적 컴포넌트
  • 컴포넌트 루트 노드

# 기본 사용법

트랜지션을 사용하는 기본 사용법을 살펴봅시다. 먼저 아래 데모를 실행해보세요.

Fade Flip X
Fade Flip Y

데모 코드는 다음과 같습니다. 코드를 살펴보면 <transition> 컴포넌트를 사용해 CSS 클래스 속성을 요소에 적용합니다.





 
 
 

 
 
 































































<template>
  <div class="demo-transition">
    <button @click="isVisible = !isVisible">트랜지션 토글</button>
    <!-- x축 페이드 플립 트랜지션 -->
    <transition name="fade-flipX">
      <div class="box" v-show="isVisible">Fade Flip X</div>
    </transition>
    <!-- y축 페이드 플립 트랜지션 -->
    <transition name="fade-flipY">
      <div class="box" v-show="isVisible">Fade Flip Y</div>
    </transition>
  </div>
</template>

<script>
export default {
  name: "DemoBasicTransition",
  data: () => ({
    isVisible: true
  })
};
</script>

<style scoped>
.fade-flipX-enter-active,
.fade-flipX-leave-active {
  transition: all 1s;
}
.fade-flipX-enter,
.fade-flipX-leave-to {
  opacity: 0;
  transform: rotateX(360deg);
}

.fade-flipY-enter-active,
.fade-flipY-leave-active {
  transition: all 1s 0.2s;
}
.fade-flipY-enter,
.fade-flipY-leave-to {
  opacity: 0;
  transform: rotateY(360deg);
}

.box {
  width: 150px;
  padding: 20px;
  margin: 20px auto 0;
  background: #3dbd83;
  color: #fff;
  font-weight: bold;
}

button {
  cursor: pointer;
  border: 1px solid #ccc;
  border-radius: 4px;
  background: #fff;
  padding: 1em;
  color: #434343;
  font-size: 12px;
  transition: all 0.3s ease-in;
}
button:hover,
button:focus {
  border-color: currentColor;
}
button:focus {
  outline: none;
}
button:active {
  transform: scale(0.95);
}
</style>

# 트랜지션 클래스

<transition> 컴포넌트는 진입(enter)과 진출(leave) 흐름에 따라 트랜지션을 적용합니다.

# 진입(enter)

  1. v-enter
    진입 시작 상태로 요소가 삽입되기 전에 추가됩니다. (keyframe 1 이후 제거)

  2. v-enter-active
    진입 활성 상태로 트랜지션 중에는 유지되다가 종료 시점에서 제거됩니다. (keyframe 2 삽입)
    ※ 활성 클래스에 트랜지션, 애니메이션을 설정합니다. (duration, delay, easing curve)

  3. v-enter-to
    진입 종료 상태로 트랜지션 종료 시점에 제거됩니다. (keyframe 2 삽입)

# 진출(leave)

  1. v-leave
    진출 시작 상태로 진출 트랜지션이 트리거 될 때 삽입되었다가 제거됩니다. (keyframe 1 이후 제거)

  2. v-leave-active
    진출 활성 상태로 트랜지션 중에는 유지되다가 종료 시점에서 제거됩니다. (keyframe 2 삽입)
    ※ 활성 클래스에 트랜지션, 애니메이션을 설정합니다. (duration, delay, easing curve)

  3. v-leave-to
    진출 종료 상태로 진출 트랜지션이 트리거 될 때 삽입되었다가 종료 시점에 제거됩니다. (keyframe 2 삽입)


NOTE

v- 접두사는 name 속성이 없는 <transition> 컴포넌트를 사용할 경우 적용되는 기본 값 입니다.

<transition> ... </transition>
1
.v-enter { ... }
.v-enter-active { ... }
.v-enter-to { ... }
.v-leave { ... }
.v-leave-active { ... }
.v-leave-to { ... }

<transition name="fade-flipX">
  <div class="box" v-show="isVisible">Fade Flip X</div>
</transition>

CSS Transition 속성은 활성(active) 상태 클래스에 적용해야 합니다. 그리고 트래지션의 시작/끝(to) 점에 모션 스타일을 적용합니다.


 
 




 
 




/* transition 속성은 활성 상태 클래스에 적용해야 합니다. */
.fade-flipX-enter-active,
.fade-flipX-leave-active {
  transition: all 1s;
}

/* 트랜지션의 모션은 트랜지션 시작/끝 점에 적용합니다. */
.fade-flipX-enter,
.fade-flipX-leave-to {
  opacity: 0;
  transform: rotateX(360deg);
}

진입/진출 트랜지션을 개별적으로 적용하고 싶을 경우는 활성 클래스를 나눠 스타일을 적용합니다.


 


 










/* 진입,진출 활성 상태에 개별 transition 속성을 적용 합니다. */
.slide-fade-enter-active {
  transition: all .3s ease;
}
.slide-fade-leave-active {
  transition: all .8s cubic-bezier(1.0, 0.5, 0.8, 1.0);
}

/* 트랜지션 모션은 동일한 코드를 사용해 교차 점을 동일하게 만듭니다. */
.slide-fade-enter,
.slide-fade-leave-to {
  transform: translateX(10px);
  opacity: 0;
}
