# 컴포넌트란?

Vue 컴포넌트는 재사용 가능한 Vue 인스턴스 입니다. 기본 제공되는 표준 HTML 요소를 확장하여 재사용 가능한 코드를 캡슐화하여 사용할 수 있습니다. 컴포넌트는 Vue의 컴파일러에 의해 동작이 추가된 사용자 정의 요소(Custom Element) 입니다.


# 컴포넌트 트리

컴포넌트 시스템은 Vue 애플리케이션을 구성하는 핵심 개념입니다. 애플리케이션의 영역을 구분하여 재사용할 수 있도록 코드 블록을 나눠 개발할 수 있습니다. 컴포넌트는 다른 컴포넌트를 포함할 수 있어 DOM 트리와 마찬가지로 컴포넌트 트리를 구성합니다.
